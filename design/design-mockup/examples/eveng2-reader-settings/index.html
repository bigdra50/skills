全 4 画面 (List / Detail / Reading / Settings) のミニチュアをノードとして配置、遷移を矢印で接続。 各 frame は実機サイズ準拠で操作可能、viewport プリセット (375 / 393 / 768 / 1024 / 1920 …) で auto-layout の挙動も一括確認。下部 States Matrix で各画面の全 variants を網羅。

将来の汎用化メモ: settings-overview.html の flow.nodes / flow.edges 構造を JSON スキーマ化 + flow-graph.js に切り出すと、任意プロジェクトの画面遷移可視化ツールになる。
