Flow Overview — Interactive

eveng2-reader 全画面 (List → Detail → Reading + Settings) の遷移と状態を俯瞰。

LIVE — 全 frame が操作可能
viewport (CSS px, w × h)

▸ User Flow (4 screens)

左上 List がエントリ。歯車 → Settings、カード → Detail、Detail の「読む」 → Reading。 矢印は遷移発火時に点灯。各 frame 内の値変更も即時反映 (dropdown は閉じても消えません)。

▸ States Matrix

各画面の全 variant を網羅。各 frame は独立 state を持ち操作可能。 画面遷移系 (card click / gear / 読む / back) は Matrix では発火しません — 状態変化のみ反映。

使い方

  • viewport プリセット: 実機の CSS px (w×h) を切替 → frame 自体の外形が viewport に追従。auto-layout で並びも再計算される
  • frame 内スクロール: コンテンツが viewport 高さを超えると縦スクロール (実機と同じ挙動)
  • ※ デスクトップ域 (1280/1920) は実画面サイズだが、ページ全体に対して大きすぎるため max-height: 80vh で制限
  • List ノード: ⚙ → Settings、📋 → クリップボード貼付シミュ、カードクリック → Detail
  • Detail ノード: ★ → fav トグル、ステータスボタン → 状態変更、「読む」 → Reading、← → List
  • Reading ノード: ← → Detail に戻る
  • Settings ノード: select 変更 → preview-bar 即更新、⟳ → 3 (default) リセット、← → List
  • RESET ALL → 全 frame の state を初期化 (viewport は維持)
